Lo-Dash v2.4.1

Web Builder

See the Lo-Dash custom builds documentation for more information.
All options do not require any quotations except for the iife option.

Build a strict build, with ES5 strict mode enabled.

Use the category command to pass comma separated categories of functions to include in the build.
Valid categories (case-insensitive) are “arrays”, “chaining”, “collections”, “functions”, “objects”, & “utilities”.

Use the exports command to pass comma separated names of ways to export the lodash function.
Valid exports are “amd”, “commonjs”, “global”, “node”, & “none”.

Use the iife command to specify code to replace the IIFE that wraps Lo-Dash.

Use the include command to pass comma separated names of functions to include in the build.

Use the minus command to pass comma separated function/category names to remove from the build.

Use the plus command to pass comma separated function/category names to add to the build.

Use the moduleId command to specify the AMD module ID for Lo-Dash.
